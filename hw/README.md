# ハードウェアモジュール - Raspberry Pi & LED制御

**担当** ：HWリード  
**役割** ：ラズパイセットアップ・カメラ接続・LED制御

## 実装内容

- Raspberry Pi OS セットアップ
- カメラ接続・設定

- GPIO制御・爆速点滅実装

## 必要な部品

- LED（赤色 5mm）
- トランジスタ（2N2222）
- 抵抗（220Ω、10kΩ）
