# AIモジュール - 顔認識実装

**担当** ：画像AI  
**役割** ：顔検出・認識・判定ロジック実装

## 実装フロー

### Phase 1: 環境構築（Week 1）
- OpenCV インストール
- mediapipe インストール
- Jupyter Notebook で動作確認

### Phase 2: 顔検出実装（Week 1-2）
- mediapipe で顔検出パイプライン構築
- Webカメラ（またはラズパイカメラ）からのフレーム取得
- リアルタイム顔検出のデモ実装

### Phase 3: 顔認識モデル学習（Week 2）
- 事前学習済みモデル（FaceNet、VGGFace2）のダウンロード
- 既知人物のデータセット作成（各人物20-30枚）
- 顔埋め込み抽出・保存
- JSON/pickle で「既知人物DB」を作成

### Phase 4: 判定ロジック実装（Week 2）

顔検出 → 埋め込み抽出 → DB比較 → 閾値判定

### Phase 5: テスト・最適化（Week 2-3）
- 異なる照度・角度での精度テスト
- 閾値の調整
- 処理速度の最適化（FPS向上）
